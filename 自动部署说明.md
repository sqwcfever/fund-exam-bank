# 基金从业资格题库 - 自动部署说明

## 方式一：自动部署脚本（推荐）

### 前置要求
- 安装 Node.js：https://nodejs.org/zh-cn/download/

### 部署步骤

#### Windows 用户：

**方法1：双击运行（最简单）**
1. 打开 `基金从业资格考试三科题库` 文件夹
2. 双击运行 `deploy.bat`
3. 按提示操作即可

**方法2：命令行运行**
```bash
cd 基金从业资格考试三科题库
.\deploy.bat
```

**方法3：PowerShell（推荐）**
```powershell
cd 基金从业资格考试三科题库
.\deploy.ps1
```

### 首次部署提示

脚本会自动：
1. ✓ 检查 Node.js 是否安装
2. ✓ 自动安装 Surge 部署工具
3. ✓ 部署到 Surge.sh

**首次部署需要：**
- 输入邮箱（任意格式，如：your@email.com）
- 设置密码（记住即可）

### 部署成功

会显示访问链接，例如：
```
https://fund-exam-bank-1234.surge.sh
```

手机浏览器打开即可使用！

---

## 方式二：手动部署（备用）

如果自动脚本失败，可以手动部署：

### 1. 安装 Node.js
访问 https://nodejs.org/zh-cn/download/ 下载安装

### 2. 安装 Surge
```bash
npm install -g surge
```

### 3. 部署
```bash
cd 基金从业资格考试三科题库
surge
```

按提示输入邮箱、密码、域名即可。

## 方式三：其他平台

### Vercel
1. 访问 https://vercel.com
2. 用邮箱注册
3. 点击 "Add New Project"
4. 选择 "Import Git Repository" 下方的 "Continue"
5. 直接拖拽文件夹上传

### Cloudflare Pages
1. 访问 https://dash.cloudflare.com/sign-up
2. 邮箱注册
3. 进入 Pages → Create a project
4. 直接上传文件夹

## 常见问题

**Q: 部署失败怎么办？**
A: 
- 检查网络连接
- 重新运行脚本会生成新域名
- 尝试手动部署

**Q: 如何更新网站？**
A: 修改文件后重新运行部署脚本即可

**Q: 数据会丢失吗？**
A: 不会，做题记录保存在浏览器本地存储中

**Q: 可以自定义域名吗？**
A: 可以，在部署时输入你想要的域名，如：`my-exam.surge.sh`
