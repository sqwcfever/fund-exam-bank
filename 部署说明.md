# 基金从业资格题库 - 部署说明

## 方式一：Netlify Drop（推荐，最简单）

### 步骤：
1. 打开 https://app.netlify.com/drop
2. 将整个 `基金从业资格考试三科题库` 文件夹拖入网页
3. 等待上传完成，自动获得访问链接（如 `https://xxx-xxx-xxx.netlify.app`）
4. 用手机浏览器打开链接即可使用

**优点**：无需注册，30秒完成部署

---

## 方式二：GitHub Pages（永久免费）

### 步骤1：注册GitHub账号
1. 访问 https://github.com
2. 点击 **Sign up**
3. 输入邮箱、设置密码、创建用户名
4. 完成验证

### 步骤2：创建仓库
1. 登录后点击右上角 **+** → **New repository**
2. **Repository name** 填：`fund-exam-bank`
3. 选择 **Public**（公开）
4. 点击最下方 **Create repository**

### 步骤3：上传文件
1. 在新仓库页面点击 **uploading an existing file** 链接
2. 点击 **choose your files**
3. 选择以下4个文件上传：
   - index.html
   - styles.css
   - app.js
   - exam_database.js
4. 点击最下方 **Commit changes**

### 步骤4：启用Pages
1. 点击仓库顶部 **Settings** 标签
2. 左侧菜单点击 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. **Branch** 选择 **main**，文件夹选 **/(root)**
5. 点击 **Save**

### 步骤5：访问
- 等待1-2分钟
- 访问地址：`https://你的用户名.github.io/fund-exam-bank/`
- 手机浏览器打开即可使用

---

## 方式三：Gitee Pages（国内访问快）

### 步骤：
1. 访问 https://gitee.com 注册账号
2. 创建仓库，名称：`fund-exam-bank`
3. 上传4个文件
4. 进入 **服务** → **Gitee Pages**
5. 点击 **启动**
6. 获得访问链接

**优点**：国内访问速度快

---

## 文件清单

部署只需要这4个文件：

## 常见问题

**Q: 部署后数据会丢失吗？**
A: 不会，所有做题记录保存在手机浏览器本地

**Q: 可以分享给朋友用吗？**
A: 可以，把链接发给朋友即可

**Q: 如何更新题库？**
A: 重新上传新的 `exam_database.js` 文件

**Q: 手机访问卡顿？**
A: 题库数据较大（约5MB），首次加载需要等待，之后缓存到本地
